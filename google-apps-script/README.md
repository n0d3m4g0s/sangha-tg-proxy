# Google Forms + Apps Script: пошаговая инструкция

## 1. Создать Google Форму

1. Открой https://forms.google.com
2. Нажми **"Пустая форма"** (Blank)
3. Назови форму: **"Telegram Proxy — Sangha"**
4. В описание формы вставь:

```
Proxy для доступа к Telegram в России.

После отправки формы вам на почту придёт персональная ссылка.
Откройте её на телефоне — Telegram предложит подключить прокси.

Ссылка персональная — пожалуйста, не передавайте её другим людям.
Если ссылка попадёт к посторонним, она будет отозвана и вам выдадут новую.

Работает на всех устройствах (телефон, компьютер, планшет).
Настройка занимает 1 нажатие.
```

5. Добавь 3 поля:

   **Поле 1:** Email
   - Нажми на значок настроек (шестерёнка) вверху формы
   - Поставь галочку **"Collect email addresses"** (Собирать адреса электронной почты)
   - Это автоматически добавит поле Email

   **Поле 2:** Telegram username или телефон
   - Тип: Short answer (Текст краткий ответ)
   - Название: `Telegram username или телефон`
   - Описание: `Без @. Примеры: username или 79263131331`
   - Обязательное: Да

   **Поле 3:** Кодовое слово
   - Тип: Short answer (Текст краткий ответ)
   - Название: `Кодовое слово`
   - Обязательное: Да

## 2. Привязать к Google Таблице

1. В форме нажми вкладку **"Responses"** (Ответы) вверху
2. Нажми зелёную иконку Google Sheets (или "Link to Sheets")
3. Выбери **"Create a new spreadsheet"** (Создать новую таблицу)
4. Назови: `Sangha Proxy Users`
5. Нажми **"Create"**

Таблица откроется. Она будет автоматически получать ответы из формы.

## 3. Добавить колонки для автоматизации

В таблице после последнего заполненного столбца добавь заголовки:

| (авто) Timestamp | (авто) Email | (авто) Telegram username | (авто) Кодовое слово | Secret | Link | Status | Activated |
|---|---|---|---|---|---|---|---|

Первые 4 колонки заполняются автоматически из формы. Последние 4 — скриптом.

**Важно:** Посмотри номера колонок (A=1, B=2, ...) и убедись что они совпадают с `COL` в скрипте. Если Email в колонке B (2), username в C (3), кодовое слово в D (4) — тогда:
- Secret = E (5)
- Link = F (6)
- Status = G (7)
- Activated = H (8)

Это совпадает с настройками по умолчанию в `Code.gs`.

## 4. Установить Apps Script

1. В таблице зайди в **Extensions → Apps Script**
2. Откроется редактор скриптов
3. Удали всё содержимое файла `Code.gs`
4. Скопируй и вставь содержимое файла `Code.gs` из этой папки
5. **Обязательно измени настройки** в блоке `CONFIG`:
   - `API_URL` — адрес твоего API (например `https://nl.rigpa.space`)
   - `API_KEY` — значение `SANGHA_API_KEY` из файла `.env`
   - `CODE_WORD` — кодовое слово (по умолчанию `ригпа`)
   - `ADMIN_EMAIL` — твой email для уведомлений об ошибках
6. Нажми **Ctrl+S** (сохранить)

## 5. Установить триггер

1. В редакторе Apps Script запусти функцию `createTrigger`:
   - Выбери `createTrigger` в выпадающем списке функций (рядом с кнопкой Run)
   - Нажми **Run** (треугольник)
2. Google попросит разрешения — нажми **"Review Permissions"**
3. Выбери свой аккаунт
4. Нажми **"Advanced"** → **"Go to Sangha Proxy (unsafe)"**
5. Нажми **"Allow"**

Это нужно сделать ОДИН раз. После этого скрипт будет автоматически запускаться при каждом новом ответе на форму.

## 6. Проверить

1. Открой форму (кнопка "Preview" — глаз вверху)
2. Заполни тестовый ответ:
   - Email: твой email
   - Username: `test_check`
   - Кодовое слово: `ригпа`
3. Отправь форму
4. Проверь:
   - В таблице должна появиться строка с заполненными колонками Secret, Link, Status
   - На email должно прийти письмо с ссылкой на прокси
5. Попробуй нажать на ссылку в письме — Telegram должен предложить подключить прокси

## Устранение проблем

**Скрипт не срабатывает:**
- Проверь что триггер установлен: Apps Script → Triggers (часы слева) — должен быть `onFormSubmit`
- Проверь логи: Apps Script → Executions (слева)

**Ошибка "API returned 403":**
- Проверь что `API_KEY` в скрипте совпадает с `SANGHA_API_KEY` в `.env`

**Ошибка "API returned 5xx":**
- Проверь что API работает: `make health`
- Проверь логи: `make logs-api`

**Email не приходит:**
- Проверь папку спам
- Лимит Gmail: ~100 писем в день для бесплатного аккаунта, ~1500 для Workspace
